About the Plugin

Sahar Animated Grid | Elementor Image Grid is an Elementor addon that lets you create different animated image grids for your website. There are different grid types available. Horizontal grid, vertical grid and diagonal grid. The images are animated based on scroll amount so the grid is a new and creative one that grabs users attention at a first look. There are also lots of options available like grid speed, grid degree, grid height, grid images, gap size etc. The image grid is a great and unique way of showing your images, gallery, portfolio etc.
