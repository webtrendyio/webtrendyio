About the Plugin

Go Night Dark Mode/Night Mode WordPress Plugin enables dark mode or night mode with multiple options for your website. Dark mode is a new trend on web design. Desktop apps like Adobe Softwares have already implemented dark mode into their UI, now this plugin adds dark mode/night mode feature into your WordPress installation.

Plugin Options:

Detect OS Color Scheme
Set dark mode based on system configurations.
Time Based Dark Mode
Automatically turn dark mode on certain hours.
Display Switch Above Pages
Display dark mode toggle button above all pages.
Dispaly Switch in Menu
Display dark mode switch in site menu.
Default Dark Mode
Set dark mode as default mode for your site.
Color Palette
choose colors from 12 available color palettes.
Exclude Category
Exclude certain categories from dark mode.
Exclude Pages
Exclude certain pages from dark mode.
Exclude Elements
Exclude certain elements from dark mode.
Button Style
Choose the type of the toggle button. (15 Styles)
Button Size
Set the size of the button.
Custom Button Text
Set the label of the button.
Button Margin
Set the margin (space around) the button.
Button Location
Choose the location of the button on the page.
Display Switch Above Posts
Display dark mode toggle button above posts.
Remember Dark Mode
Set the mode that user selected the last time he/she visited your site.
Backend Dark Mode
Add dark mode to the backend (admin panel) of your website.
Shortcode
Add the button to the location you want via inserting the shortcode.
Custom Colors
Set your custom colors for backgrounds, text, input, buttons, links etc.
Replace Images in Dark Mode
Replace some images in dark mode. (e.g. a light version of your logo)
Custom CSS
Write your own CSS code to be applied only in dark mode.
Dim Image Brightness
Dim the brightness of images in dark mode.
