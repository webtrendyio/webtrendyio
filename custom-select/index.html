Use it with any style

Options:

bgColor: Set Background Color

hoverColor: Set Hover Color

fontSize: Set Font Size

listHeight: Set the Height of Dropdown

Example:

					new CustomSelect({
						bgColor: "#fff",
						hoverColor: '#eee',
						fontSize: '3em',
						listHeight: '200px'
					});
