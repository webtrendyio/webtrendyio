WP Animated Buttons is a WPBakery Page Builder addon with a beautiful and trendy set of animated buttons. You can define buttons type, animation, color and link. How it works?

01
Install the plugin

After downloading the plugin, go to your WordPress dashboard, install and activate the plugin.

02
Add via WPBakery

Add WP Animated Buttons to the page via WPBakery Page Builder's add element section.

03
Refresh your site

Now go to front end of your website and refresh the page to see the result.
