WP Post Filter View

Let users decide how to see your content!

About the plugin

WP Post Filter View is a WPBakery addon that enables view filters (list/grid) for posts and custom posts. You can filter the view of your posts based on different options like small grid, large grid or list. The plugin is super lightweight and easy to use. Just install the plugin and add it to your page via WPBakery page builder.

How it works?

01
Install the plugin

After downloading the plugin, go to your WordPress dashboard, install and activate the plugin.

02
Add via WPBakery

Add WP Post Filter View to the page via WPBakery Page Builder's add element section.

03
Refresh your site

Now go to front end of your website and refresh the page to see the result.

Why use it?

A very important subject in UX is letting users customize your site as much as possible. This plugin allows user to select how to see your content. That will definitly increase user engagement and user experience of your site.
